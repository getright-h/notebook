## 包装类型

通过包装类型，可用把基本类型转换为**对象**

1. `String()`

2. `Number()`

3. `Boolean()`

   1. ```js
      let foo = new Number(3)
      ```



玩家不要自用，浏览器自动执行：

当我们对一些基本类型的值去添加方法和属性时，浏览器会**临时**用包装类型把它转为对象，然后再调用对象上的方法和属性

```js
var s1 = "some text"; 
var s2 = s1.substring(2);
```

浏览器做的事：

1. 创建 String 类型的一个实例；

2. 在实例上调用指定的方法；

3. 销毁这个实例；

   ```js
   //相当于内部执行了如下：
   var s1 = new String("some text"); 
   var s2 = s1.substring(2); 
   s1 = null;
   ```
