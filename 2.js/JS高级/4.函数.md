**立即执行函数iife**

*也称匿名函数自调用*

```js
(fn(){
 
 })()
```

隐藏实现

不会污染全局命名空间

编写js模块，向外暴露函数

```js
(function () {
    function foo() {
        console.log("foo");
    }

    window.$ = function () {
        return { foo: foo };
    };
})();

$().foo();

```



**构造函数**

1. 通过`new`关键字来创建的构造函数，相较于工厂模式，不需要通过`return 值`来返还函数
2. `new`关键字给我们隐式地创建了一个对象并返还值
3. 构造函数的`this`指向它的创建者
