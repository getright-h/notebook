#### 3.下一页按钮

1. 判断是否为最大页

   1. 不是。当前页+1 ，且显示的条目为

      ```js
      let start = (nowPage - 1) * 5;
                      // let  let end = nowPage * 5 - 1;
      ```

   2. 是。当前页不变，且显示的条目为

      ```
      let start = (nowPage - 1) * 5;
      let end = (nowPage - 1) * 5 - 1 + (student.length % 5)
      ```
